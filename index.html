Neovim for Cygwin

This is my personal Cygwin package overlay server hosting neovim for Cygwin and the dependencies that are not yet present or not recent enough in Cygwin:

  • neovim
  • luajit
  • luajit-lpeg
  • luajit-mpack
  • msgpack-c
  • libluv
  • unibilium
  • libtermkey
  • libvterm

If things go well I am willing to maintain them on the Cygwin repository. Until then it lives here so it can be tested as easily as possible.

If you're using these packages I would like any kind of feedback, even if things are working well for you. Please don't hesitate to open an "issue" on Github!

Usage

Download the Cygwin installer from cygwin.com and run it from a command line or a batch file using the following arguments:
  • -s https://mirrors.kernel.org/sourceware/cygwin/ This is a package server with the official Cygwin packages (you can choose any other mirror),
  • -s https://cygwin.paktolos.net This is my package overlay server, and
  • -K https://kgraefe.paktolos.net/AA73A441.bin This is the public key used to verify the integrity of my packages.

The full command line looks like that: setup-x86_64.exe -s https://mirrors.kernel.org/sourceware/cygwin/ -s https://cygwin.paktolos.net -K https://kgraefe.paktolos.net/AA73A441.bin

Now you can select neovim from the package list which also installs all the dependencies.

Note: The Cygwin installer remembers the package sites but not the public key. Therefore if the setup is run again without any options it will fail to verify my package overlay server and therefore fail in general. So you must use the command line arguments above for consecutive runs (use a Batch file!) or deselect my server in the list of download sites (but then you won't get updates).
